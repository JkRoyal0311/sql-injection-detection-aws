# SQL Injection Detection and Prevention using AWS

## Overview
This project demonstrates detection and prevention of SQL Injection attacks
using a multi-layered security approach on AWS Cloud.

## Features
- Secure authentication
- SQL Injection prevention
- AWS WAF protection
- Real-time monitoring and alerts

## Tech Stack
- Backend: Java / Python
- Database: MySQL
- Cloud: AWS (EC2, RDS, WAF, CloudWatch, SNS)
